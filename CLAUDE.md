# CLAUDE.md — orion-sdr-view

## Project

`orion-sdr-view` is a keyboard-driven SDR spectrum visualization tool in Rust (crate name `orion-sdr-view`, bin name `orion-sdr-view`, v0.0.5), edition 2024, built on egui/eframe. Displays live spectrum, persistence density, and scrolling waterfall from a configurable signal source.
